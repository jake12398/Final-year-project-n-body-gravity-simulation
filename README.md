# Final-year-project-n-body-gravity-simulation
This project simulates the gravitational interaction of multiple stars using a classical N-body physics model. It models motion under Newtonian gravity and tracks system evolution over time.

## Features
- N-body gravitational force calculation
- Velocity-Verlet integration scheme
- Adaptive time stepping
- Energy calculations (kinetic + potential)
- Binary system interactions
- Snapshot saving for simulation analysis
- Visualisation of star positions over time

## Physics Model
- Newtonian gravity
- Softening parameter to avoid singularities
- Conservation of energy tracking
- Centre-of-mass frame velocity correction

## Tech Stack
- Python
- NumPy
- Matplotlib

## Outputs
- Star positions over time
- Snapshot files for simulation states
- Energy and system dynamics tracking
